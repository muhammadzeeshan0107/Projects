# 🍽️ One on One Palace

A modern and responsive restaurant website for **One on One Palace** — 
a neighborhood restaurant with real character.

👉 [one-on-one-palace.netlify.app](https://one-on-one-palace.netlify.app/)

## 🛠️ Built With

- HTML5
- CSS3

## ✨ Features

- 🏠 Home — Hero section with restaurant branding
- 🍕 Menu — Food items display section
- 🏢 About Us — Restaurant story and values
- 📅 Reservation — Table booking section
- 📞 Contact — Contact information
- 📱 Fully Responsive (Mobile Friendly)
- 🎨 Warm and elegant design with cream & brown theme
